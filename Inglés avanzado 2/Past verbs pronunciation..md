## When I pronounce *ed*?.
I pronounce *ed* only with the verbs end in */t/*-*/d/*$\Longrightarrow$*/ed/*.
For example:

| Verb | Past simple |
| --- | --- |
| want | want**ed** |
| start | start**ed** |
| need | need**ed** |
| end | end**ed** |
